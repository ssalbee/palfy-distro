Palfynux Official Repository

A repó hozzáadásához appendáld a következőt a /etc/pacman.conf konfigurációs fájlba:

[palfynux]
Server = http://easimer.tk:8080/palfynux/repo/$arch

Palfynux alaprendszer telepítése:

# pacman-key --init ## Kulcstartó inicializálása
# pacman-key --populate archlinux ## Kulcstartó megtöltése mesterkulcsokkal
# dirmngr < /dev/null
# pacman-key -r BC07F417 ## Kulcs hozzáadása a kulcstartóhoz
# pacman-key --lsign-key BC07F417 ## Kulcs helyi aláírása
# pacman -Syuu
# pacman -S palfynux

A csomag tartalmazza a következő programokat:

  • LibreOffice
  • Evince PDF olvasó
  • Code::Blocks
  • Python
  • Fritzing
  • Arduino IDE
  • Chromium
  • PuTTY
  • VLC Media Player
  • Icarus Verilog
  • KiCAD
  • Eagle
  • OpenSSH

Hamarosan érkezik még: LXDE, Tina, MicroC és az ALTERA IDE.
